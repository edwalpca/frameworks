PHP-FrameWork 1.5 - What's New?

Versión 1.5 ( Actualizado 26-06-2011)

Nuevas características y mejoras:

  • php-barcode: Integración del php-barcode, para la generación de código de barras, para ver ejemplo clic aquí.se agregó una funcion dentro de la libreria "php-barcode.php" para guardar la generacion de codigo de barras en formato .jph de forma local en el servidor como una imagen, para hacer el embedbed a un PDF o manipular el codigo de barras como se desee.
  • xlsx: Se integra la librería PHPExcel para la lectura de archivos de Excel version 2007, con esta librería puedes realizar
    lectura de un archivo y exportar en este mismo formato, para ver ejemplo ver aquí. documentación del API, solo funciona con PHP 5.2 o mayor.
  • FckEditor 2.6.4.1: integración del fckeditor para editar html, ver ejemplo aquí, para ver como se usa o integra clic aquí.
  • jquery-tooltip:Libreria de gran utilidad por sus efectos, son ayudas tipo "ToolTip", para ver ejemplo aquí.
  • lightbox: Libreria para efectos de imagenes o de utilidad para galeria de fotos.
  • php-sql: se corrigió la librería recordset.inc.php, para que inteprete código PHP dentro de los SQL, la version anterior interpretaba unicamente codigo PHP dentro de la funcion listado, con esta version en su .sql puedes meter código por medio de la sintaxis concact ('@php $var = 1 + 2') as resultado, ver ejemplo aqui
  • pclzip: Libreria para Manipular archivos .zip ( ver utilizada en el archivo llamado: "pclzip.lib.php".
  • Fileupload: Libreria para subir archivos al servidor de una manera mas facil y con control de archivo. Ver utilidad en el archivo llamado "class.fileupload.inc.php".
  • Sajax: Manejo de la libreria Sajax.php, para ver ejemplo de como utilizar revisar la carpeta "sajax_examples" dentro de js.
